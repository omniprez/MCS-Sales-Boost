MCS Sales Boost - Dashboard API Fix

Current Status

This page injects a fix for the dashboard API to ensure data loads correctly.

Before using this fix, your dashboard showed error messages for all data.

The fix works by:

  1. Intercepting dashboard API requests
  2. Returning mock data if the API call fails
  3. Providing realistic sample data for all dashboard components

How to Use

Click the button below to add the fix to your browser. Then open your dashboard page.

Technical Details

This fix adds a script that intercepts fetch requests to the dashboard API endpoints:

/api/dashboard/revenue-summary
/api/dashboard/pipeline-summary
/api/dashboard/sales-leader
/api/dashboard/leaderboard
/api/dashboard/win-rate
/api/dashboard/conversion-rate
/api/dashboard/gp-summary
/api/dashboard/quota-completion
/api/dashboard/avg-deal-size
/api/dashboard/pipeline-distribution
/api/dashboard/revenue-trend

When these endpoints return 404 errors, the script provides mock data instead.
